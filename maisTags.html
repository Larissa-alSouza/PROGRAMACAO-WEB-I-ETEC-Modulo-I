Aqui é o começo da página

A tag "p" representa um espaço especial para um paragrafo

A tag "input" serve para o usuário digitar textos

A tag "button" serve para realizar ações dentro da página

A tag img serve para colocar imagem dentro das páginas html

a tag "div" serve para criar espaços estratégicos dentro das páginas HTML


A tag "br" significa quebra de linha
dentro de um texto
